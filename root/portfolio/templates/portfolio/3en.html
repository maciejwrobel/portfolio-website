2048 Game

2048 is a single-player sliding block puzzle game designed by Italian web developer Gabriele Cirulli. The objective of the game is to slide numbered tiles on a grid to combine them to create a tile with the number 2048, however, one can continue to play the game after reaching the goal, creating tiles with larger numbers. It is played on a 4x4 grid using the arrows. Every time you press a key - all tiles slide. Tiles with the same value that bump into one-another are merged.

This version was developed using Python and Tkinter module. As a relaxing addition, music plays in the background, and the performer is my own band.
